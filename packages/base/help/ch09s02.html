9.2. Transfer Sessions with Core Archives

9.2.1. Export a Core Archive

The Core Archive Export function packs all the resources along with a session CML file into a compressed "Core ARchive", a ".car" file. It will contain all the images, datasets, annotations and their attachments.

To transfer a complete Corelyzer project to another machine, use the File > Core Archive > Export... menu item.

Enter a name for the project.

Figure 9.1. Name the project for export


Choose a name and location for the exported .car file.

Figure 9.2. Choose name and destination


The progress bar of the session window will fill as the archive is exported, followed by a success message.

Figure 9.3. Success!


The exported .car file can be transfered to other machines and shared with colleagues.

9.2.2. Import a Core Archive

To import a core archive on the destination machine, use the File > Core Archive > Import... menu item.

Select the .car file when prompted.

Figure 9.4. Select the .car file to import


Corelyzer will extract its contents and load them into the current session.
