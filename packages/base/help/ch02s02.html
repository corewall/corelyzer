2.2. Install Corelyzer

2.2.1. Microsoft Windows Users

Extract the downloaded .zip archive by right-clicking the file and selecting "Extract All...". Extract Corelyzer to the desired directory and double-click the Corelyzer icon to launch.

2.2.2. Apple Mac OS X/macOS Users

Unzip the downloaded file by double-clicking and you'll see the application icon.

You can drag the application to your preferred disk location, like the Applications directory or your Desktop. Double click the icon to launch Corelyzer.
